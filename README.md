# IOTA Street Brawler

On-chain Tamagotchi fighter built on IOTA Move.

## Features

- On-chain player progression
- Staking arena matches
- Weapon marketplace
- Web3 wallet integration via IOTA dApp Kit

## Stack

- React + Vite
- IOTA dApp Kit
- IOTA Move smart contracts
- TypeScript

## Setup

```bash
npm install
npm run dev
